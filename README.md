# Dev Productivity Tracker

This project consists of a backend server and a mobile application.

## Project Structure

- `backend/`: Express.js server with PostgreSQL database integration.
- `mobile/`: React Native (Expo) application for tracking developer contributions.

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. `npm run dev`

### Mobile
1. `cd mobile`
2. `npm install`
3. `npx expo start`

## Prerequisites
- Node.js
- PostgreSQL (for backend)
- Expo Go app (for mobile testing)
